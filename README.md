# Sistema de Gerenciamento em C

Projeto em C com menu interativo para cadastro e listagem de usuários.

## Estrutura
- `src/` → arquivos .c
- `include/` → arquivos .h
- `Makefile` → para compilar facilmente

## Como compilar
No terminal MSYS2 MinGW64, dentro da pasta do projeto:

```bash
make
./sistema
